# 🎯 Jogo do Número Secreto

Este projeto é um jogo simples desenvolvido com **HTML, CSS e JavaScript**, onde o jogador precisa adivinhar um número secreto entre **1 e 10**.

## 📌 Sobre o projeto

O jogo gera aleatoriamente um número secreto, e o jogador deve tentar descobri-lo digitando números até acertar.
Durante as tentativas, o jogo informa se o número secreto é **maior ou menor** que o número escolhido, ajudando o jogador a chegar à resposta correta.

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## 🎮 Como jogar

1. Digite um número entre **1 e 10** no campo indicado.
2. Envie sua tentativa.
3. O jogo informará se o número secreto é **maior ou menor**.
4. Continue tentando até **acertar o número correto**.
